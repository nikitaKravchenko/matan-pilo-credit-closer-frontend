## Starting

Git clone project
```
git clone ssh://git@bitbucket.devit-team.com/~kirill.nefodov/credit_admin_app.git

cd ./credit_admin_app/
```

Raise the database
```sh
make build
```
Run in dev mode
```sh
make install

make dev
```
